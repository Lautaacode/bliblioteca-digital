# Sistema de gestion de Libreria Digital

## Descripcion 
Aplicacion diseñada en Python usando 
programancion orientada a objetos
para gestion de libros, usuarios y prestamos

## Funcionalidad

- CRUD Libros
- CRUD Usuarios
- Registro de prestamos
- Registro de retornos
- Prestamos activos

## Conceptos POO

- Herencia
- Polimorfismo
- Agregación
- Composición
- Decorador personalizado
- Metaclase
- Patrón Singleton
- Patrón Factory

## Iniciar Aplicacion

Para iniciar la aplicacion, se debe ejecutar el archivo main.py

```bash
python main.py
```
